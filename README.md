# 📊 Sales Dashboard — Superstore Analysis

An interactive sales performance dashboard built in **Tableau**, using a cleaned version of the popular Superstore dataset. The dashboard visualizes key business metrics like sales, profit, and regional performance to help identify trends and opportunities.

## 🔍 Overview

This project involved cleaning raw sales data and building a Tableau dashboard to explore:
- Overall sales and profit trends
- Regional and category-wise performance
- Top/bottom performing products
- Customer segment analysis

## 🛠️ Tools Used

- **Tableau** — dashboard design and visualization
- **Excel** — data cleaning and preparation

## 📁 Files

| File | Description |
|------|-------------|
| `TASK-1 (SALES DASHBOARD).twb` | Tableau workbook containing the full interactive dashboard |
| `Superstore_Cleaned_for_Tableau.xlsx` | Cleaned dataset used as the data source for the dashboard |

## 🚀 How to View

1. Download [Tableau Public](https://public.tableau.com/) or Tableau Desktop (free)
2. Open `TASK-1 (SALES DASHBOARD).twb`
3. Explore the interactive filters and visualizations
